# 🛒 Ultima Tech - Premium E-commerce Ecosystem

### Live Link : https://ultima-tech-ecommerce-niib.vercel.app/

**Ultima Tech** is a cutting-edge, full-stack e-commerce platform designed for tech enthusiasts. Built with **Next.js 15**, **MongoDB**, and **NextAuth.js**, it provides a seamless shopping experience and a powerful administrative suite for managing high-end gadgets.

---

## ✨ Key Features

- **Dynamic Product Browsing**: Advanced filtering by categories and real-time search functionality.
- **Secure Authentication**: Integrated **NextAuth.js** for robust user sessions and protected dashboard access.
- **Inventory Management**: A comprehensive Admin Dashboard to add, view, and remove products.
- **Smart Data Protection**: Custom server-side logic prevents the deletion of the first 6 essential products to maintain site integrity.
- **Persistent UI State**: Category filters persist through page reloads using URL search parameters.
- **Modern UI/UX**: Built with **Tailwind CSS** and **Lucide React** for a sleek, responsive, and dark-themed aesthetic.
- **Interactive Feedback**: Enhanced user notifications using **SweetAlert2** for successful operations and restricted actions.
