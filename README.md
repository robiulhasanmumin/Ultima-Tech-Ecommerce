# 🛒 Ultima Tech - Premium E-commerce Ecosystem

### 🌐 Live Link: [https://ultima-tech-ecommerce-niib.vercel.app/](https://ultima-tech-ecommerce-niib.vercel.app/)

![Ultima Tech Banner](/websiteImg.png) 
 

**Ultima Tech** is a high-performance, full-stack e-commerce platform built for the modern gadget enthusiast. Leveraging **Next.js 15**, **MongoDB**, and **NextAuth.js**, it delivers a seamless transition from product discovery to secure checkout and order management.

---

## ✨ Advanced Features

### 🛍️ Core Shopping Experience
- **Dynamic Product Browsing**: Instant filtering by categories and real-time search with persistent URL state.
- **Smart Wishlist System**: Users can save their favorite gadgets to a personalized wishlist with real-time sync in the Navbar.
- **Seamless Ordering**: A streamlined checkout process where users can specify quantities and shipping details effortlessly.

### 📑 Post-Purchase & Management
- **Order Management**: Users can view their order history, update shipping details (before processing), or cancel orders with double-confirmation safety.
- **Instant Invoice Generation**: Automated, professional invoices for every purchase to enhance user trust and record-keeping.
- **Admin Command Center**: A robust dashboard to manage inventory (Add/Edit/Delete) with protected logic for core products.

### 🔐 Security & UX
- **Secure Authentication**: Integrated **NextAuth.js** for protected routes and personalized user sessions.
- **Interactive Feedback**: Premium notifications via **SweetAlert2** and real-time UI updates using **SWR**.
- **Dark-First Aesthetic**: A sleek, responsive UI built with **Tailwind CSS** and **Lucide React** icons.

## 🛠️ Tech Stack

| Layer | Technology |
| :--- | :--- |
| **Framework** | [Next.js 15 (App Router)](https://nextjs.org/) |
| **Database** | [MongoDB](https://www.mongodb.com/) |
| **Auth** | [NextAuth.js](https://next-auth.js.org/) |
| **State/Fetching** | [SWR (Stale-While-Revalidate)](https://swr.vercel.app/) |
| **Styling** | [Tailwind CSS](https://tailwindcss.com/) |
| **Dialogs** | [SweetAlert2](https://sweetalert2.github.io/) |
